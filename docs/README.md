1. `CHANGE_LOG.md` — aktueller Stand und letzte Änderungen
2. `STATUS.md` — Übersicht und offene Punkte
3. `CONTEXT_RECOVERY.md` — Schnelleinstieg neue Session

| Pfad | Inhalt |
