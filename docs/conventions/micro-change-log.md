# Micro-Change-Log — Pflichtformat

**Gilt für:** Jede Änderung an Code, Config, Doku, Infra — egal wie klein.

## Pflichtformat je Eintrag

```
#### HH:MM — [Ziel in einem Satz]
- **Dateien:** `pfad/datei.ext`
- **Aktion:** [Was gemacht wurde, 1–2 Sätze]
- **Ergebnis:** [OK | FEHLER: Beschreibung]
- **Verifikation:** [Wie geprüft]
- **Nächster Schritt:** [Was als Nächstes]
- **Blocker:** [– | Beschreibung]
- **Weiter bei:** [Datei oder Aufgabe]
```

## Arbeitsrhythmus

| Phase | Pflicht |
|-------|---------|
| Vor Änderung | Ziel + Dateien notieren |
| Nach Änderung | Ergebnis + Verifikation eintragen |
| Bei Unterbrechung | „Weiter bei:" eintragen |
| Bei Fehler | Diagnose + Fix dokumentieren |
