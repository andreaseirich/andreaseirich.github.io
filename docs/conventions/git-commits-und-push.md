# Git: Commits und Push

## Commits: klein, logisch getrennt

- **Nach jeder in sich schlüssigen Änderung:** committen
- **Ein Commit = ein logisches Thema**
- Commit-Message: `type: kurzbeschreibung`

## Push

- Zeitnah pushen nach Commits einer Arbeitsphase
- Secrets nie committen
